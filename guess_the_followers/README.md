Guess the Followers

A simple command-line Python game where players compare two popular personalities and guess which one has more social media followers. The game continues until the player makes an incorrect guess.

Features

- Randomly selects two personalities for comparison
- Displays each personality's name, description, and country
- Player guesses which personality has more followers
- Keeps track of the player's score
- Ends the game when an incorrect guess is made
- Automatically generates a new comparison after each correct answer

How It Works

Two personalities are displayed on the screen. Based on the information provided, the player guesses which personality has more social media followers by choosing A or B. If the guess is correct, the score increases and the game continues. An incorrect guess ends the game and displays the final score.

How to Run

1. Clone or download this repository.
2. Open the project in your preferred code editor.
3. Run "main.py".
4. Compare the two personalities.
5. Enter A or B to make your guess.
6. Continue playing to achieve the highest score possible.
