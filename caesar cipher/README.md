Caesar Cipher

A simple command-line Python application that encrypts and decrypts text using the Caesar Cipher algorithm. The program allows users to encode or decode messages by shifting each letter of the alphabet by a user-defined number of positions.

Features

- Encrypt any text using a custom shift value
- Decrypt encrypted messages
- Supports uppercase and lowercase letters
- Preserves spaces, numbers, and special characters
- Handles shift values larger than 26 using modulo arithmetic
- Option to continue encrypting or decrypting multiple messages without restarting the program

How It Works

The Caesar Cipher is one of the oldest encryption techniques. Each letter in the message is shifted by a fixed number of positions in the alphabet.

Example:

- Original: "HELLO"
- Shift: "3"
- Encrypted: "KHOOR"

How to Run

1. Clone or download this repository.
2. Open the project in your preferred code editor.
3. Run "main.py".
4. Choose whether to encrypt or decrypt a message.
5. Enter your text and shift value.
6. View the output.
