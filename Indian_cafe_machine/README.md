# Indian Cafe Machine

A simple command-line Python application that simulates an Indian café. Users can order popular Indian beverages, and the program checks ingredient availability, processes payments, and prepares the selected drink.

## Features
- Choose from multiple Indian beverages
- Calculates the cost of each drink
- Accepts coin-based payments
- Returns change when applicable
- Checks ingredient availability before preparing a drink
- Updates available resources after each order
- Displays a report of remaining ingredients and total profit
- Allows users to turn the machine off

## Menu
- Tea
- Coffee
- Masala Tea
- Filter coffee

## How to Run
1. Clone or download this repository.
2. Open the project in your preferred code editor.
3. Run `Indian_cafe_machine.py`.
4. Select your preferred beverage.
5. Insert coins when prompted.
6. Enjoy your drink if sufficient resources and payment are available.
