Hangman Game

A simple command-line Hangman game built with Python. The player must guess a hidden word one letter at a time before running out of lives.

Features

- Randomly selects a word for each game
- Letter-by-letter guessing
- Tracks correct and incorrect guesses
- Displays remaining lives
- Prevents repeated guesses
- Win and lose conditions
- Option to play again

How It Works

The program randomly chooses a word from a list. The player guesses one letter at a time. If the guessed letter is correct, it is revealed in the word. If it is incorrect, the player loses one life. The game ends when the player guesses the entire word or runs out of lives.

How to Run

1. Clone or download this repository.
2. Open the project in your preferred code editor.
3. Run "main.py".
4. Guess one letter at a time.
5. Try to reveal the complete word before your lives reach zero.
